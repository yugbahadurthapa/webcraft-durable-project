# webcraft-durable-project
This project is a responsive business website built to help small businesses establish a quick and professional online presence. AI tools like Qwen were used for ideation, content planning, and improving clarity. The project focuses on clean design, usability, and fast deployment using modern no-code tools.
## Alibaba Cloud Deployment Status

This project does not include a backend deployment on Alibaba Cloud.

The application is a frontend-focused project built using Durable Website Builder.
AI assistance (Qwen) was used for content ideation and planning only.

No Alibaba Cloud services or APIs were integrated during this build.
