# webcraft-durable-project
This project is a responsive business website built to help small businesses establish a quick and professional online presence. AI tools like Qwen were used for ideation, content planning, and improving clarity. The project focuses on clean design, usability, and fast deployment using modern no-code tools.
## Alibaba Cloud Deployment Note

This project was designed to be deployed on Alibaba Cloud using services such as OSS and Qwen AI.
However, due to the unavailability of an India region on Alibaba Cloud at the time of submission,
a full live deployment could not be completed.

To ensure transparency, this repository includes:
- Alibaba Cloud OSS configuration files
- Deployment-ready structure
- Architecture diagram showing intended Alibaba Cloud usage

The project follows Alibaba Cloud best practices and can be deployed instantly
in any supported region.
